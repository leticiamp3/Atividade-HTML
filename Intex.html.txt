  <h1>🚀 O Futuro do Desenvolvimento Web</h1>
  <p> O desenvolvimento web está em constante evolução, trazendo novas tecnologias e desafios. Criar um site moderno envolve conhecer diferentes linguagens e práticas recomendadas.<p>
    <h2> 📌 Principais Tecnologias Utilizadas</h2>
    <p>O desenvolvimento web pode ser dividido em três camadas principais:<p>
<ol>
  <li><strong>Frontend</strong> - Responsável pela interface visual. Tecnologias principais:
  <ul>
    <li> HTML (estrutura)

      <li> CSS (estilo)

        <li> JavaScript (Interatividade)
          </ul>
  <li><strong>Backend</strong> - Responsável pelo funcionamento interno do site. Principais tecnologias:
  <ul>
    <li>Node.js
      <li>Python (Django, Flask)
        </ul>
        <li><strong>Banco de dados</strong>  - Onde as informações são armazenadas. Alguns exemplos:

<ul>
  <li>MySQL

    <li> PostgreSQL

      <li>MongoDB
        
        </ul>
        </ol>
        <h2>
          🔥 Tendências do Desenvolvimento Web </h2>
          <p>Algumas tendências estão moldando o futuro da web:<p>
            <ul>
              <li><strong>Aplicativos web progressivas (PWA)</strong> - Sites que funcionam como apps nativos.
              <li><strong> Inteligência artificial </strong> - Personalização de experiências com IA.
              <li><strong>Web 3.0 e Blockchain</strong> - Tecnologias descentralizadas para segurança e privacidade.
              </ul>
